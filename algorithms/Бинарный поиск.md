> `Бинарный поиск` - алгоритм, на вход которого поступает **отсортированный** список, при нахождении требуемого элемента возвращается его позиция, иначе null.

Принцип поиска: каждый раз происходит переход в середину диапазона, тем самым отсекая половину оставшихся чисел с определенной стороны, и так пока не найдется требуемый элемент.

```
При бинарном поиске числа из списка [1...100], элемент определяется за 7 шагов. При поиске в [1...240e3], элемент находится за 18 шагов
```

> Для списка из _n_ элементов, бинарный поиск выполняется за _log2n_, за _логарифмическое время_

> Бинарный поиск работает только если список отсортирован

Бинарный поиск в `python`

```python
def binary_search(arr, target):
"""
Классический алгоритм бинарного поиска
Поиск индекса заданного элемента, в отсортированном массиве

Бинарный поиск работает только с отсортированным массивом
Эффективно находит элемент, каждый раз отбрасывая половину массива

Сложность O(log₂(n))

:param arr: заданный массив
:param target: заданный элемент
"""

	# границы поиска
	low = 0
	high = len(arr)-1
	
	# пока находимся в границах поиска
	while low <= high:
		# середина диапазона
		mid = (low + high) // 2
		guess = arr[mid]
		
		# найден в середине
		if guess == target:
			return mid
		# меньше середины - ищем в левой половине
		elif guess > target:
			high = mid - 1
		# больше середины - ищем в правой половине
		else:
			low = mid + 1
	# элемент не найден
	return None
```

```python
# проверка
arr = [1, 3, 5, 7, 9]
print(binary_search(arr, 5))
# на выходе выдаст 2
```
